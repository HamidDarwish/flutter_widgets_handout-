Project Description

This project demonstrates the basic structure of a Flutter application using different widgets.
Each section focuses on one widget and explains its functionality through code and output.


---

Explanation of Each Part

1️) Scaffold

Purpose:
Provides the main structure of the app.

What it does:

Creates a full screen layout

Includes AppBar and Body

Displays the text "Hello Hamid Darwish" in the center of the screen



---

2️) AppBar

Purpose:
Displays the top navigation bar.

What it does:

Shows the title "Student App"

Centers the title using centerTitle: true

Changes the background color to indigo



---

3️) FloatingActionButton (FAB)

Purpose:
Represents a primary action button.

What it does:

Adds a circular button with a plus (+) icon

When pressed, prints "Clicked" in the console



---

4️) Drawer (Navigation Menu)

Purpose:
Provides a side navigation menu.

What it does:

Adds a drawer that slides from the left

Contains:

A header with the title "Menu"

Menu items: Home and Settings


Can be opened using swipe or menu icon



---

5️) Bottom Navigation Bar

Purpose:
Allows navigation between different sections.

What it does:

Creates two tabs:

Home

Profile


Assigns icons to each tab

Sets the default selected tab to index 0



---

6️) Body (Column Layout)

Purpose:
Displays content in the center of the screen.

What it does:

Uses a Column to arrange multiple widgets vertically

Displays two texts:

"Student App"

"Welcome students"


Adds spacing using SizedBox

Aligns content to the center

Conclusion

This project helps students understand:

The role of each Flutter widget

How to build a simple UI layout

How different components (AppBar, Drawer, FAB, etc.) work together


It serves as a strong foundation for building more advanced mobile applications.
